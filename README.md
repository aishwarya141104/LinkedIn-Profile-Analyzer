# AI-Powered LinkedIn Profile Analyzer Using Gemini

## Project Overview

The AI-Powered LinkedIn Profile Analyzer is a Python-based application that uses Google Gemini AI to evaluate a candidate's LinkedIn profile and generate a professional improvement report.

The system analyzes important LinkedIn profile sections and provides actionable recommendations to improve profile visibility, recruiter discoverability, and professional branding.

The final analysis report is automatically generated and exported as a PDF document.

---

## Features

* Analyze LinkedIn Headline
* Evaluate About Section
* Review Skills Section
* Analyze Experience Section
* Suggest Recruiter-Friendly Keywords
* Check Resume Alignment
* Generate Overall LinkedIn Score
* Provide Professional Recommendations
* Export Analysis Report as PDF

---

## Technologies Used

* Python
* Google Gemini API
* Google Colab
* ReportLab
* Generative AI

---

## Input Parameters

Users provide:

* LinkedIn Headline
* About Section
* Skills
* Experience

---

## Output

The system generates:

* Headline Quality Score
* About Section Score
* Skills Evaluation
* Experience Evaluation
* Recruiter Keyword Suggestions
* Resume Alignment Feedback
* Overall LinkedIn Score
* Improvement Recommendations

The report is saved as:

LinkedIn_Report.pdf

---

## Project Workflow

1. User enters LinkedIn profile information.
2. Gemini AI analyzes the profile.
3. AI generates detailed feedback and recommendations.
4. Python processes the response.
5. ReportLab creates a professional PDF report.
6. User downloads the final report.
